# Vector Pulse – DDoS Detection using Machine Learning

## Overview

Vector Pulse is a Machine Learning–based cybersecurity system developed to detect Distributed Denial of Service (DDoS) attacks in network traffic. The system analyzes traffic patterns and identifies anomalies that may indicate malicious activity, helping enhance network security.

## Key Features

* Detects abnormal and suspicious network traffic
* Identifies potential DDoS attacks
* Uses Machine Learning models for prediction
* Supports continuous network monitoring
* Scalable for large datasets

## Technologies Used

* Programming Language: Python
* Domain: Cybersecurity & Machine Learning
* Dataset: Network Traffic Dataset

### Libraries

* Pandas – Data preprocessing and analysis
* NumPy – Numerical computations
* Scikit-learn – Model training and evaluation

## System Workflow

1. Data Collection – Gather network traffic data
2. Data Preprocessing – Clean and normalize dataset
3. Feature Extraction – Extract important features
4. Model Training – Train ML algorithms
5. Traffic Analysis – Analyze incoming data
6. Attack Detection – Classify traffic as normal or attack
